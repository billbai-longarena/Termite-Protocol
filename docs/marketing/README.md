## How to use

- Use the launch posts when announcing a new release or sharing the project in communities.
- Use the article drafts for a blog, newsletter, Substack, Medium, 知乎, 公众号, or technical forum post.
- Use the homepage copy when building a project site, landing page, or docs homepage.

## Positioning reminders

- Lead with the problem: stateless agents lose continuity.
- Then explain the mechanism: coordination through the environment.
- Then show proof: audits, commits, and quality improvement.
- Keep Termite Commander positioned as optional automation, not the protocol itself.
