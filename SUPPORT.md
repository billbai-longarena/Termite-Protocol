# Support

## Where to ask what

- **Usage questions**: GitHub Discussions
- **Bug reports**: GitHub Issues
- **Feature requests**: GitHub Issues or Discussions, depending on scope
- **Security concerns**: `SECURITY.md`

## Before asking for help

- Read `README.md` and `QUICKSTART.md` first.
- Include your operating system, shell, and the exact command you ran.
- Include relevant files or output such as `.birth`, `BLACKBOARD.md`, and signal paths when safe to share.

## What maintainers need to debug quickly

- The install command you used
- Whether you are testing a fresh project or upgrading an existing colony
- The output from `./scripts/field-arrive.sh`
- The latest release tag or commit SHA you are using

## Response expectations

Best effort. Questions with clear reproduction steps and context are much easier to answer quickly.
